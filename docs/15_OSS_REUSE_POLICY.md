# 15 OSS Reuse Policy

Permissive dependencies are allowed only with review.

Learn-only by default:

- Dify
- Typebot
- Firecrawl
- Legacy FlowAI backend

Do not copy AGPL, FSL, custom-license, or legacy code into this project without explicit review.

## Dependency Addition Requirements

Document:

- Why needed
- Alternatives considered
- License
- Maintenance risk
- Security risk
- Boundary where it will be used

OSS idea reuse must be documented in the relevant task or decision log.
