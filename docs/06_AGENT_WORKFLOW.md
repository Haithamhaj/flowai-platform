# 06 Agent Workflow

## Protocol

1. Discover: read `AGENTS.md`, project context, active task, and relevant shard.
2. Skill/MCP Readiness: check `docs/17_SKILL_MCP_READINESS.md`.
3. Plan: state scope, likely files, acceptance checks, tool needs, and risks.
4. Confirm: wait when approval is required.
5. Implement: keep changes small and task-scoped.
6. Test: run relevant tests/build or document why not run.
7. Review: apply `docs/08_REVIEW_CHECKLIST.md`.
8. Handoff: update task notes and provide final summary.

## Prohibited Behaviors

- “I just implemented everything.”
- Broad refactors.
- Adding dependencies without justification.
- Skipping the Skill/MCP Readiness Check.
- Changing architecture silently.
- Hiding failing tests.
- Claiming untested AI/RAG/channel behavior works.
- Copying code from the old repo or unsafe OSS sources.
